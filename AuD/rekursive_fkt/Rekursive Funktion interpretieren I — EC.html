Rekursive Funktion interpretieren I

Einreichungszeitraum beendet: 10.11.2014 08:00
Eine Ebene höher

Es ist folgende Funktion gegeben: 
Algorithm: f(x,y)
Input:     x,y ganzzahlig
Output:    ?
   if y = 0 then x
            else f(x-1, y-1)
   fi
  1. Beschreiben Sie die Auswertung an mehreren Beispielen. Was berechnet diese Funktion? Für welche Werte terminiert der Algorithmus nicht?
  2. Überlegen Sie wie der Algorithmus zu erweitern ist, damit er für alle ganzzahligen Werte terminiert. Schreiben Sie dafür eine Java-Methode

    public static int rek(int x, int y).

  3. Formulieren Sie nun Ihren verbesserten Algorithmus als iterative Methode

    public static int iter(int x, int y).
