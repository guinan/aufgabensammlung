class ComplexNumber

Einreichungszeitraum beendet: 27.01.2015 22:00
Eine Ebene höher

Mathematik I ist Voraussetzung
Schreiben Sie eine Klasse ComplexNumber, wobei der Real- und Imaginärteil vom Datentyp double sind.
  • Neben einem geeigneten Konstruktor
    public ComplexNumber(double re, double im)
    und den Methoden
    public ComplexNumber add(ComplexNumber k)
    und
    public ComplexNumber mult(ComplexNumber k)
    soll auch der Betrag
    public double absoluteValue()
    einer komplexen Zahl ermittelt werden.
  • Erweitern Sie die Klasse um eine Methode
    public ComplexNumber sqrt() throws InvalidInputForSquareRoot,
    die dann die Quadratwurzel der komplexen Zahl zurückgibt, wenn der Imaginärteil 0 ist. Andernfalls soll eine Ausnahmebedingung InvalidInputForSquareRoot ausgelöst werden.
  • In der main-Methode sollen Instanzen der Klasse ComplexNumber erzeugt und alle Methoden getestet werden.
Hinweis: Die Methode 
  public String toString() {
    // bitte nicht veraendern (wegen Backend-Kontrolle)
    return real + " + " + imag + "*i";
  }
muss in dieser Form wegen der automatischen Kontrolle in Ihrer Einreichung enthalten sein!
